# 📊 Vrinda Store Sales Analysis Dashboard

## Overview

The Vrinda Store Sales Analysis Dashboard is an Excel-based data analytics project designed to analyze sales performance, customer behavior, and order trends. The dashboard transforms raw sales data into meaningful insights using Pivot Tables, Pivot Charts, Slicers, and interactive visualizations.

## Business Problem

Vrinda Store generates sales data from multiple customers, products, and sales channels. Analyzing this data manually is time-consuming and makes it difficult to identify trends, top-performing categories, and customer preferences. This dashboard helps stakeholders make informed business decisions through data-driven insights.

## Objectives

* Analyze overall sales performance.
* Identify top-performing products and categories.
* Understand customer demographics and purchasing behavior.
* Evaluate sales performance across different states.
* Compare sales channels and order distribution.
* Generate actionable business insights.

## Key Performance Indicators (KPIs)

* Total Sales Revenue
* Total Orders
* Order Status Analysis
* Gender-wise Sales Contribution
* Age Group Analysis
* State-wise Sales Performance
* Channel-wise Sales Distribution
* Category-wise Sales Analysis

## Dashboard Features

* Interactive Dashboard using Slicers
